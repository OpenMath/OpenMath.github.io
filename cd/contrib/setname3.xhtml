The function_set operator generates the set of functions between the sets specified as its arguments.

function_set

Description:

The function_set operator generates the set of functions between the sets specified as its arguments.

cf Hom(A,B) in Category theory, and mapsto in the sts CD.

For a set of n-argument functions, function_set will take n+1 arguments, specifying the sets of the n arguments and the range.
