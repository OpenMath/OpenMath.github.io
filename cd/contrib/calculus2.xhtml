This CD provides a vocabulary for talking about integrals.

variable_of_integration

Description:

This symbol represents the variable with respect to which an integral is calculated.

Example: 
  <OMOBJ xmlns="http://www.openmath.org/OpenMath">
    <OMA>
      <OMS cd="relation1" name="eq"/>
      <OMA>
        <OMS cd="calculus2" name="variable_of_integration"/>
        <OMA>
          <OMS cd="calculus1" name="int"/>
          <OMBIND>
            <OMS cd="fns1" name="lambda"/>
            <OMBVAR>
              <OMV name="x"/>
            </OMBVAR>
            <OMA>
              <OMS cd="transc1" name="sin"/>
              <OMV name="x"/>
            </OMA>
          </OMBIND>
        </OMA>
      </OMA>
      <OMV name="x"/>
    </OMA>
  </OMOBJ>
<math xmlns="http://www.w3.org/1998/Math/MathML">
 <apply><csymbol cd="relation1">eq</csymbol>
  <apply><csymbol cd="calculus2">variable_of_integration</csymbol>
   <apply><csymbol cd="calculus1">int</csymbol>
    <bind><csymbol cd="fns1">lambda</csymbol>
     <bvar><ci>x</ci></bvar>
     <apply><csymbol cd="transc1">sin</csymbol><ci>x</ci></apply>
    </bind>
   </apply>
  </apply>
  <ci>x</ci>
 </apply>
</math>
eq (variable_of_integration (int (lambda [ x ] . (sin ( x) ) ) ) , x)
calculus2.variable_of_integration(calculus1.int(fns1.lambda[$x -> sin($x)])) = $x
variable_of_integration ⁡ ( ∫ sin ⁡ ( x ) d x ) = x
