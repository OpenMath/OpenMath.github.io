length

Role:
application
Description:

This symbol represents a function whose argument should be a list. It returns the length of its argument.

Example:
The length of the list [1,2,3] is 3. 
<OMOBJ xmlns="http://www.openmath.org/OpenMath" version="2.0">

       <OMA><OMS cd="list3" name="length"/>
            <OMA><OMS cd="list1" name="list"/>
                 <OMI> 1 </OMI>
                 <OMI> 2 </OMI>
                 <OMI> 3 </OMI>
            </OMA>
       </OMA>
       <OMI> 3 </OMI>
  </OMA>
</OMOBJ>
<math xmlns="http://www.w3.org/1998/Math/MathML">
 <apply><csymbol cd="relation1">eq</csymbol>
  <apply><csymbol cd="list3">length</csymbol>
   <apply><csymbol cd="list1">list</csymbol>
    <cn type="integer">1</cn>
    <cn type="integer">2</cn>
    <cn type="integer">3</cn>
   </apply>
  </apply>
  <cn type="integer">3</cn>
 </apply>
</math>

list_selector

Role:
application
Description:

This symbol takes a positive integer n and a list, and represents the n-th element of that list.

Example:

    <OMI> 2 </OMI>
    <OMA>
      <OMS name="list" cd="list1"/>
      <OMV name="a"/>
      <OMV name="b"/>
      <OMV name="c"/>
    </OMA>
  </OMA>
</OMA>
</OMOBJ>

select

Description:

This symbol takes two lists as arguments, L and M say. The second argument is a list containing only entries from [1..n], where n is the length of L. The symbol represents the function which returns a list whose length is equal to the length of M, and having at position k the value of L at position M_k.

Example:

append

Role:
application
Description:

This symbol represents a function with two arguments, the first of which should be a list. When applied to A and b, it represents the list obtained from A by appending the element b to it.

Commented Mathematical property (CMP):
Appending b to the list constructed from the first element a and the rest lst1 gives the same list as the list constructed from the first element a and the rest the list obtained from appending b to lst1.
Formal Mathematical property (FMP):

concatenate

Role:
application
Description:

The operation of joining one list to another

Commented Mathematical property (CMP):
Concatenating b to the list constructed from the first element a and the rest lst1 gives the same list as the list constructed from the first element a and the rest the list obtained from concatenating b to lst1.
