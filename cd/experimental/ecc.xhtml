Pair

Role:
application
Description:

The pairing constructor. It takes two OpenMath objects as first element and second element of the pair, and a third optional OpenMath object that represents the type of the pair.

Example:

PairProj1

Role:
application
Description:

The first projection function that extracts the first component of a Pair. It satisfies the sigma-reduction rule.

Example: 
	<OMOBJ xmlns="http://www.openmath.org/OpenMath" version="2.0" cdbase="http://www.openmath.org/cd">
	<OMA><OMS cd="relation1" name="eq"/>
	<OMA><OMS cd="ecc" name="PairProj1"/>
	<OMA><OMS cd="ecc" name="Pair"/>
	     <OMI>0</OMI>
             <OMI>1</OMI>
	</OMA>
	</OMA>
	<OMI>0</OMI>
	</OMA>
	</OMOBJ>
<math xmlns="http://www.w3.org/1998/Math/MathML">
 <apply><csymbol cd="relation1">eq</csymbol>
  <apply><csymbol cd="ecc">PairProj1</csymbol>
   <apply><csymbol cd="ecc">Pair</csymbol><cn>0</cn><cn>1</cn></apply>
  </apply>
  <cn>0</cn>
 </apply>
</math>
eq (PairProj1 (Pair (0, 1) ) , 0)
ecc.PairProj1(ecc.Pair(0, 1)) = 0
PairProj1 ⁡ ( Pair ⁡ ( 0 , 1 ) ) = 0

Tuple

Role:
application
Description:

The n-ary tupling constructor when n>2. The arguments are the element of the tuple. Tuple objects can also be constructed by successive nesting of Pair.

Example: 
	<OMOBJ xmlns="http://www.openmath.org/OpenMath" version="2.0" cdbase="http://www.openmath.org/cd">
	<OMA><OMS cd="relation1" name="eq"/>
	     <OMA><OMS cd="ecc" name="Pair"/>
	          <OMI>0</OMI>
	          <OMA><OMS cd="ecc" name="Pair"/>
                  <OMI>1</OMI>
	          <OMI>2</OMI>
	          </OMA>
	     </OMA>
       	     <OMA><OMS cd="ecc" name="Tuple"/>
	          <OMI>0</OMI>	
		  <OMI>1</OMI>	
		  <OMI>2</OMI>
             </OMA>
	</OMA>
	</OMOBJ>
<math xmlns="http://www.w3.org/1998/Math/MathML">
 <apply><csymbol cd="relation1">eq</csymbol>
  <apply><csymbol cd="ecc">Pair</csymbol>
   <cn>0</cn>
   <apply><csymbol cd="ecc">Pair</csymbol><cn>1</cn><cn>2</cn></apply>
  </apply>
  <apply><csymbol cd="ecc">Tuple</csymbol><cn>0</cn><cn>1</cn><cn>2</cn></apply>
 </apply>
</math>
eq (Pair (0, Pair (1, 2) ) , Tuple (0, 1, 2) )
ecc.Pair(0, ecc.Pair(1, 2)) = ecc.Tuple(0, 1, 2)
Pair ⁡ ( 0 , Pair ⁡ ( 1 , 2 ) ) = Tuple ⁡ ( 0 , 1 , 2 )
