lift_binary

Role:
application
Description:

This symbol denotes the lift of a binary operator on elements of X to a component-wise operators on subsets of X.

Example:
{1,2} + {2,3} = {3,4,5}

( lift_binary ⁡ ( + ) ) ⁡ ( { 1 , 2 } , { 2 , 3 } ) = { 3 , 4 , 5 }
Commented Mathematical property (CMP):
A lift_binary(op) B = { a op b: a in A and B in B } 
Note that, to write this as an FMP, we have to assume a universal U such that
A, B and A lift_binary(op) B are subsets of U.
