This CD holds symbols denoting various types of numbers, vectors and similar constructs. Principally these are required for compatibility with MathML.

class

Role:
attribution
Description:

A symbol to be used within an OpenMath attribute to specify the class attribute of the object. The annotation should be an OpenMath string representing the value of the class attribute.

foreign_attribute

Role:
application
Description:

A symbol to be used as the head of the OpenMath application to construct the object used as the value of the foreign attribution. The four arguments of this function should be OpenMath strings representing in order, the Namespace, prefix and local name and value of the MathML attribute.

Example:
The MathML attribute xmlns:other="http://example.com" other:att="bla" on the object x may be represented by: 
<OMOBJ xmlns="http://www.openmath.org/OpenMath" version="2.0" cdbase="http://www.openmath.org/cd">
  <OMATTR>
    <OMATP>
      <OMS cd="mathmlattr" name="foreign"/>
      <OMA>
	<OMS cd="mathmlattr" name="foreign_attribute"/>
	<OMSTR>http://example.com</OMSTR>
	<OMSTR>other</OMSTR>
	<OMSTR>att</OMSTR>
	<OMSTR>bla</OMSTR>
      </OMA>
    </OMATP>
    <OMV name="x"/>
  </OMATTR>
</OMOBJ>
<math xmlns="http://www.w3.org/1998/Math/MathML">
 <semantics>
  <ci>x</ci>
  <annotation-xml cd="mathmlattr" name="foreign">
   <apply><csymbol cd="mathmlattr">foreign_attribute</csymbol>
    <cs>http://example.com</cs>
    <cs>other</cs>
    <cs>att</cs>
    <cs>bla</cs>
   </apply>
  </annotation-xml>
 </semantics>
</math>
