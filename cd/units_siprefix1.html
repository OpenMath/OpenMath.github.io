This CD defines symbols for the SI prefixes to indicate multiplication by a power of 10.

yotta

Description:

This symbol represents the fact that the subsequent unit has been effectively multiplied by $10^24$

Commented Mathematical property (CMP):
This is unit multiplication by $10^24$
Formal Mathematical property (FMP):

    </OMA>
  </OMA>
  <OMA>
    <OMS name="times" cd="arith1"/>

</OMA>
