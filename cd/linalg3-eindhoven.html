This CD deals with basic dimension notions of a matrix and a vector.

November, 2004, this CD was created from the former CD linalg4 by removing all symbols referring to eigenvalues, eigenvectors, rank, and the characteristic polynomial. As a result, this CD deals with basic dimension notions of a matrix and a vector.

size

Role:
application
Description:

This symbol represents the function which takes one vector argument and returns the length of that vector.

Example:
The length of the vector [1,2,3] is equal to 3.

  <OMA><OMS cd="relation1" name="eq"/>
       <OMA><OMS cd="linalg3" name="size"/>
            <OMA><OMS cd="linalg2" name="vector"/>
                 <OMI> 1 </OMI>  <OMI> 2 </OMI> <OMI> 3 </OMI>
            </OMA>
       </OMA>
   <OMI> 3 </OMI>
  </OMA>
</OMOBJ>
