MONET Infrastructure
Ø Brokers
– register local services
– communicate with other brokers
– match a query to one or more services
• fuzzy matching versus exact matching
Ø Object Manager
– refer to objects via URIs
– objects may be persistent, exist inside a running
service, be stored by a cluster of services etc.
– allow access permissions to be set (e.g. for
collaborative working)
Ø Explanation and Debugging Tool
