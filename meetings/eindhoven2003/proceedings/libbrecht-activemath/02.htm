Intelligent Learning environment

  • The central data: the user-model
    • Stores user actions (reading, exercises, ...) in histories
    • Maintains an up-to-date table of estimated knowledge for each item
  • based on this a course can be built:
    • using the item relations
    • providing only the content estimated necessary to be learned
    • configured through the use of pedgagogical rules
  • automated suggestions can be provided
