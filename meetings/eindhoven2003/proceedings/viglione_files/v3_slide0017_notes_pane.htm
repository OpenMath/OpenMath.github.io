In the figure it is roughly shown the structure of a MSP page. It is mainly an HTML page, but for the mathlet  script that make calls to the Hidden.m functions. In the figure, at the beginning of the page, it is also shown the mathlet instruction that is needed to load the package Hidden.m. (i.e. the tag <% Mathlet Needs ["Hidden"]%>).
The resulting HTML page, representing one step of some exercise, will contain text and formulas, the latter generated as images in the classic mathematical formalism, produced  automatically by  WebMathematica.
