– Personal accounting by login and password: each user accesses a personal working space by login and password and then s/he can choose topics, type of exercise, theory and so on
– Menu for the selection of the topics: a menu is available for selecting topics and various types of exercises.
– Memory of the state of the exercise when the user logs out: when a user logs out because s/he wants to interrupt his/her exercise session, the system memorizes the state of the exercise so that, when the same user logs in again, s/he finds the exercise in the exact point in which s/he left it.
