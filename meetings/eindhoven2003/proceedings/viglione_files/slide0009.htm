MoMAM@th
This is the html page that the student sees after login
We can see the topics Limits…, and The Exam Session that we will see better in the next slides
For each topic there are some types of exercises
When the learner selects a type of exercise a new exercise is randomly generated.
Momam@th does not have a database containing exercises and solutions,
but they are randomly generated on the fly by the software any time that
a student requires one.
At the top of the interface there are four links.
The first one allows the learner to send an e-mail with her/his comments about Momam@th
The second one is a link to a Momamath overview document
Help opens the help on-line.
The teacher link allows the learner-teacher asynchronous interaction
By the theory button students have the possibility to consult theory pages related to the current exercise step.
