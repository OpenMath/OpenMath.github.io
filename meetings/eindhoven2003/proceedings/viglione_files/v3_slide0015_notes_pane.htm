The logical architecture of MoMAM@th is distributed on three layers: the data access layer contains the database and the file system that store all system data; the application layer where the information are elaborated according to the user’s inputs; the presentation layer that is responsible of the delivery.
MoMAM@th is mainly based on two applications, uPortal and WebMathematica©. uPortal is a free and open source application, developed in java by the JA-SIG group, and it is used to create web portals, that is. to organize the presentation of resources on the web. uPortal  lets to organize the web-page in sub-areas, called “channels”, managed independently by software.
In MoMAM@th we used the channels to collect exercises all belonging to a particular mathematical subject (such as Limits, Derivatives, etc.). Moreover these topics additionally are grouped in different sessions relative to particular branch of mathematics (e.g. Functional Analysis, Linear Algebra etc.).
