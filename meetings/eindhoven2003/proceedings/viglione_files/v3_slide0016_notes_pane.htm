The exercises appear in the exercise area and consist of a sequence of pages generated by Web-Mathematica, containing text, graphs, formulas and tools necessary to solve the specific problem. As the following figure shows, one exercise is made by a set of files.
Some of them, the MSP page and some Mathematica Packages, are fixed, identical for all the exercises. The only file different for each exercise is the "Template.m" that contains the logic for the specific exercise. This choice was adopted in order to make easier the task of the content development.
