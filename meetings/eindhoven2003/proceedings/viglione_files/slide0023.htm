To overcome the previous limitations the ideal solution would be to let the user be able to directly edit formulas in the browser using the symbolic mathematics notation. Moreover as shown before, the formulas of the exercise aren’t a simple series of mathematical symbols.
They could contain an inner structure that we will call template, with some fixed parts (read only parts) and some editable. The last should be described by an operator with some properties.
