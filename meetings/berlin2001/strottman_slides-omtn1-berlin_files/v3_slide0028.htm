A Scoping Rule for OMBIND
v Consider object OMBIND(q,v,e).  Then:
The scope of variable v is expression e.
– OM Standard is silent on this question. Add the above!
–  Neither attributes of variables nor (possibly compound)
quantifier q are inside v’s scope.
– OM Standard’s OMBind currying rules complicate
scoping rules a bit for multiple variables.
– Type expression for v not necessarily valid inside
expression e (variable name clashes).
