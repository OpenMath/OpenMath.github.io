Categorial types
v Notation due to Lambek
v Abstraction A/B – returns something of type A if
given argument of type B
– STS “mapsto(B,A)”;   vernacular  AB
– A/B/C/D : three args
v Application A·B – something of type A applied to
an argument of type B
–  STS “application(A,B)”;   vernacular         ??)
– (A/B/C/D) · D · C · B  “simplifies” to type A
v Plug in any type system (e.g. ECC) for atoms
– Hence “categorial” semantics / type logic
